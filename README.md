# typescript-cli-starter

A deadly simple starter for writing cli app with typescript.

- [arg](https://github.com/vercel/arg) for args parsing.
- `tsc` to compile
- `jest` and `ts-jest` to test typescript directly

## Commands

```bash
# add packages
$ yarn
# build cli
$ yarn build
# run test
$ yarn test
# debug as typescript
$ ts-node src/cli.ts

# run as nodejs
$ node dist/cli.js
```
